Project for Code The Dream Introduction to Programming 26.2 (Spring 2026) Course
Stephanie Jane Edwards


To check local weather for Greensboro, NC using an open API service, click "Weather" from the homepage navigation bar.

On the destination weather API page, click the first button, labeled "Check Weather" to get a readout of current temperature and weather conditions.

Click the second button, labeleled "7-Day Forecast", to receive a report of expected high temperatures and weather conditions for the next seven days, listed by date.
